# PyAlias

### INSTALL

1. clone repository\
  `$ git clone https://github.com/paulloclara/pyalias.git ~/.pyalias && python3 ~/.pyalias/install.py`

2. update terminal settings\
  restart the terminal or `$ source ~/<youbashrc>`

### RUN

`$ pyalias file_name args`

### UPDATE

`$ (cd ~/.pyalias && git pull)`
